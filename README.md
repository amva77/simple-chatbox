Simple Rule-Based Chatbot

Description
This is a simple chatbot built using Python. It uses rule-based logic to respond to user inputs.

Features
- Responds to basic user inputs like greetings
- Continuous conversation using loop
- Exit option using "bye"

Technologies Used
- Python

How to Run
1. Clone the repository:
   git clone https://github.com/your-username/simple-chatbot.git
2. Navigate to folder:
   cd simple-chatbot
3. Run the program:
   python chatbot.py

Example
You: hi  
Bot: Hello!  
You: how are you  
Bot: I'm doing great!
